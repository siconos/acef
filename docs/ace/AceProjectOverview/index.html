  • Theme : Sûreté des sustèmes informatisés (Thème 2)

Goal of the project

The goal of the project is to answer at the WP3 work package describe in the VAL-AMS project:

  • WP3: Advanced numerical analysis techniques.
    1. Developing numerical analysis techniques based on the non-smooth approach.
    2. Developement of an automatic D.A.E. and L.C.P. formulation tool.

Project proposal, main software functionalities

In the first step, it consists to develop a tool able to get a circuit equation formulation from a electrical circuit. This tool is an Automatique Circuit Equation Formulation software(ACEF).

At last, the software must be able to perform the transient analysis.

Software input

The input is an electrical circuit description. SPICE uses a Netlist. A Netlist is a file with an extension .cir contains the circuit description. ACEF uses the same Input file format.

Software output

The software output is the transient analysis.

Relation with siconos

ACEF need SICONOS to solve a Mixte Linear Competentarity Problem (MLCP).

Status

The project is currently under development.
