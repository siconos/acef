Parser library tests

It consist in checking if Netlist is correctly interpreted. Endeed Netlist contains macro must be evaluated. The program test displays the interpreted Netlist.

Transient analysis

The test consists in comparing result with a SPICE simulation.

Automatic tests

A script runs the program on a list of Netlist. For each circuit file, a file is created. This file contains the result of the parser, the automatic equation formulation and the transient analysis.
